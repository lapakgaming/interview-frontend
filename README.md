# LapakGaming – Technical Interview (FE)

Template project for **technical interview**.  
Next.js (App Router) + TypeScript with a mock API.

**Task / instructions** are not in this repo — they will be provided separately (e.g. Google Docs or during the briefing).

---

## Quick Start (before the session)

```bash
npm install
npm run dev
```

- Home: http://localhost:3000  
- Product page: http://localhost:3000/products/game-123  
- Mock API: http://localhost:3000/api/products/game-123  

Ensure all of the above work locally so you can focus on the task during the session.
